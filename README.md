# AstrBot SeerInfo Plugin

> 本项目基于[IronsBot](https://github.com/Nattsu39/IronsBot)开发。

> 查询功能不包含需要登录无头客户端的功能。

## 命令表

发送 **/帮助** 即可获取命令表

| 功能分类 |    指令格式    |         说明         |
| :------: | :------------: | :------------------: |
| 精灵相关 | 精灵 <名称/id> |   查询精灵基础信息   |
|          | 立绘 <名称/id> |  查询精灵或皮肤立绘  |
|          | 属性 <名称/id> |    查询属性克制表    |
|          | 异常 <名称/id> |   查询异常状态信息   |
| 刻印相关 | 刻印 <名称/id> |  查询刻印信息及数值  |
|          | 宝石 <名称/id> |   查询刻印宝石信息   |
| 装备相关 | 套装 <名称/id> |     查询套装信息     |
|          | 部件 <名称/id> |   查询装备部件信息   |
| 称号相关 | 称号 <名称/id> |     查询称号信息     |
| 其他功能 |    下周预告    |    获取下周预告图    |
|          |    开服查询    | 查询服务器是否已开服 |
